# Akina Portfolio

Personal site of **Akina** — onchain specialist and researcher.

Dark, cinematic landing page built with **React + Vite + TypeScript + Tailwind CSS**,
animated with **framer-motion** and **lucide-react** icons.

## Development

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # type-check + production build to dist/
npm run preview  # preview the production build
```

## Deployment

Deployed on **Vercel** (auto-builds from this repo, served at the root path).
